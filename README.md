# Library Demo app
by Alessandro G. Magnasco 2026, AGPLv3

This is a full stack demo library management app that uses Typescript, React, Express, Node, Postgres, and Docker.

### Features:
- WIP

### Roadmap:
- Admin users can add, remove, or edit books.
- Patron users can check books in or out.
- The collection is searchable by text fields or by tags.
- Tags can be generated by a LLM from a provided short description.
- Styling
- i18n en/es/fr
- installation instructions
